Oakwood Leisure Centre - Network

Network Infrastructure

  • Internet Connection (ISP) → Connects to the Firewall
  • Firewall (Cisco ASA 5508-X) → Provides security and access control
  • Core Router (Cisco ISR 4331) → Handles routing between VLANs and external networks
  • Core Switch (Cisco Catalyst 9500-24Y4C) → Central connection point for the network
  • Distribution Switches (Cisco Catalyst 9300 Series) → Layer 3 switches for segmentation
    • Access Switch 1 (Reception) – SW1 (Cisco Catalyst 9200-48P)
      • Customer Reception Terminals – Used for checking in members, booking classes, and processing payments.
      • Senior Receptionist Terminal – Has elevated privileges for managing memberships and handling financial transactions.
      • Cafe POS – Used in the café for processing payments.
    • Access Switch 2 (Office) – SW2 (Cisco Catalyst 9200-24P)
      • Managerial Office Workstations – Connected to business-critical software, including payroll and inventory management.
    • Access Switch 3 (Facilities & Security) – SW3 (Cisco Catalyst 9200-48P)
      • Maintenance Systems – Access to internal logs, security cameras, and door control systems.
      • Integrated Vending Machines – Networked for remote monitoring of stock and payments.
      • HVAC Control System – Manages heating, ventilation, and air conditioning across the centre.
      • Swimming Pool Sanitation & Control – Automated chemical regulation, filtration, and temperature management.
      • CCTV & Security System – Connected to the main network for monitoring.
  • Wireless Network Infrastructure
    • Wireless Controller (Cisco 9800-CL) → Manages and optimizes wireless connections
    • Access Points (Cisco Aironet 2802i Series)
      • AP1 – Reception Area (Cisco Aironet 2802i) → Covers check-in, customer seating
      • AP2 – Gym Floor (Cisco Aironet 2802i) → Provides connectivity for gym members
      • AP3 – Cafe (Cisco Aironet 2802i) → Wi-Fi for customers and POS terminals
      • AP4 – Office (Cisco Aironet 2802i) → Secure connection for managerial staff
      • AP5 – Pool Area (Cisco Aironet 2802i) → Supports pool maintenance systems and guests
      • AP6 – Staff Room (Cisco Aironet 2802i) → Internal staff network access
    • Wi-Fi Networks
      • Customer Wi-Fi (VLAN 20) – A segregated guest network for visitors.
      • Staff Wi-Fi (VLAN 30) – Used for internal communications, connected to staff email and work systems.
